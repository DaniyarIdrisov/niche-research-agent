# Таксономия характеристик: эпиляторы

## tweezers — Количество пинцетов

Главный спек категории. Чем больше — тем быстрее эпиляция.

Синонимы:
- «количество пинцетов»
- «пинцетов»
- «число пинцетов»

Типичные значения: 20, 24, 28, 32, 40, 60.

Должно быть в названии — главный фильтр в поиске.

## speeds — Скорости

Сколько режимов скорости.

Синонимы:
- «скоростей»
- «режимов скорости»

Типичные: 1, 2, 3.

## attachments — Сменные насадки

Эпиляторы обычно с несколькими насадками: эпиляция, бритьё, шейвер, для чувствительных зон, массажная.

Синонимы:
- «насадок в комплекте»
- «сменных насадок»

Типичные: 1, 3, 5, 7 насадок.

## wet_dry — Влажная и сухая эпиляция

Возможность использования в душе с водой и пеной (снижает боль).

Синонимы:
- «влажная и сухая эпиляция»
- «wet&dry»
- «влажная/сухая эпиляция»

Бинарное: есть / нет.

## light — Подсветка

Встроенная LED-подсветка для видимости тонких волосков.

Синонимы:
- «подсветка»
- «встроенная подсветка»

Бинарное: есть / нет.

## battery_life — Автономность

Сколько работает на одном заряде.

Синонимы:
- «время работы»
- «автономность»

Типичные: 30, 40, 50, 60 минут.

## power_type — Тип питания

Синонимы:
- «тип питания»
- «питание»

Значения: «аккумулятор» (мобильный), «от сети» (постоянная подача), «аккумулятор + сеть» (универсал — premium).

## warranty — Гарантия

Типичные: 1 год, 2 года.

## Что must-have в нише (≥80% топа)

- Количество пинцетов (100%)
- Сменные насадки ≥3 (85%)
- Скорости ≥2 (90%)
- Подсветка (80%)

## Что nice-to-have

- Влажная эпиляция (50% — премиум-фишка)
- Автономность ≥40 мин (60%)
- Универсальное питание (аккум + сеть, 40%)

## Замечание про сегментацию

Бюджетный сегмент (1500-3500 ₽) — упрощённые модели с 20-24 пинцетами, без wet&dry, питание от сети.
Средний (3500-7000 ₽) — 28-40 пинцетов, подсветка, аккумулятор.
Премиум (7000+ ₽) — 40-60 пинцетов, wet&dry, набор насадок 5-7 шт, премиум-бренды (Philips, Braun).
