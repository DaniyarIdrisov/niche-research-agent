# Требования к карточке товара малой бытовой техники (МБТ) на Wildberries

## Обязательные поля карточки

| Поле | Требование |
|---|---|
| Название | До 100 символов. Структура: `Категория + Бренд + Модель + ключевая характеристика`. Пример: «Фен для волос Philips BHD300 1600 Вт ионизация». |
| Бренд | Должен совпадать с производителем по документам. Бренды «No name», «Без бренда» допускаются, но снижают доверие. |
| Описание | До 5000 символов. Первые 250 символов важнее всех — они индексируются в поиске. |
| Категория | Выбирается из дерева WB. Для МБТ дерево: Бытовая техника → Красота и здоровье → … |
| Характеристики | Заполняются из списка WB. Для МБТ обязательно: мощность, тип питания, гарантия, страна производства. |
| Фото | Минимум 3, рекомендуется 7-10. Первое фото — главное (см. photo_standards.md). |
| Цена | В рублях, без знака валюты. |
| Габариты упаковки | Длина / ширина / высота в см, вес в кг. От этого зависит логистика. |

## Стоп-слова и запреты

- Нельзя использовать слово «лучший», «№1», «единственный» без подтверждения (премодерация может заблокировать).
- Запрещены сравнения с конкурентами по бренду («лучше чем Philips»).
- Нельзя обещать медицинский эффект (для электрических зубных щёток — нельзя писать «лечит дёсны», можно «массирует»).
- Нельзя использовать названия чужих брендов в описании, кроме совместимости («совместим с насадками Oral-B»).

## Длина текста

- **Название** 50-100 символов — оптимально для CTR.
- **Описание** 1500-3000 символов — оптимально для индексации поиском. Не лей воду.
- **Преимущества** (буллеты в карточке) — 5-7 пунктов, каждый до 100 символов.

## Что попадает в поиск WB

1. Название карточки (вес ×3).
2. Бренд + модель.
3. Поле «Характеристики» (структурированные значения индексируются как ключевые слова).
4. Первые 250 символов описания (вес ×2).
5. Остальное описание (вес ×1).

Из этого следует: ключевые слова, по которым вы хотите ранжироваться, должны быть в названии **и** в первых 250 символах описания **и** в характеристиках.

## Что НЕ влияет на поиск

- Фотографии (только на CTR в выдаче).
- Раздел «О бренде».
- Видео.
- Отзывы (на ранжирование влияет рейтинг и количество, не текст).
