# Skill: unit_economics

## Когда использовать

Когда нужно прикинуть, **сколько остаётся продавцу с единицы** при заданной розничной цене. Используется в PRD Writer для проверки target_price и в Reporter — для риск-секции.

## Входы

| Поле | Тип | Описание |
|---|---|---|
| `retail_price` | int | Цена на витрине, ₽ |
| `cost_price` | int | Себестоимость единицы, ₽ |
| `weight_kg` | float | Вес для расчёта логистики |
| `category` | str | Внутренний слаг — определяет комиссию WB |

## Выходы

```python
{
  "retail_price": 2500,
  "wb_commission_rub": 575,        # 23% от retail_price для МБТ
  "logistics_rub": 67,             # доставка склад→ПВЗ, средняя
  "acquiring_rub": 50,             # 2% эквайринг
  "cost_price": 1200,
  "margin_rub": 608,               # сколько остаётся продавцу
  "margin_share": 0.243,           # 24.3%
}
```

## Ставки WB (2025) для МБТ

| Категория | Комиссия |
|---|---|
| Электроника / МБТ (FBO, FBS) | ~23% |
| Логистика МБТ малой/средней массы | ~50-90 ₽ за единицу |
| Эквайринг | 1.5-2% |

**Внимание:** ставки WB регулярно меняются — для prod использования нужно подтягивать с их API. В этом скилле зашиты значения на момент написания, см. `src/tools/unit_economics.py::WB_RATES_2025`.

## Запуск

```python
from src.tools.unit_economics import calc_unit_economics

result = calc_unit_economics(
    retail_price=2500, cost_price=1200, weight_kg=0.3, category="electric_toothbrush"
)
```

## Edge cases

- **margin_share < 0** (продаём в минус) → возвращаем как есть, не подкрашиваем. Reporter сам пометит риск.
- **Незнакомая категория** → дефолтная комиссия 23% (МБТ-средняя).
